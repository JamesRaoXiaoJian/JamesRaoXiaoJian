# Pinned Repository Descriptions

These are concise GitHub repository descriptions for the pinned repositories on the profile page. They are meant for the repository "About" description field, not for README content.

| Repository | Suggested description |
| --- | --- |
| `safesentinel` | Multimodal campus safety AI with person detection, tracking, action recognition, and VLM-based risk analysis. |
| `g1_motion_player` | Unitree G1 CSV motion replay tool with C++ `rt/arm_sdk` execution, FastAPI upload API, validation, and safe transition logic. |
| `g1_mujoco_sim` | Lightweight MuJoCo preview tool for replaying Unitree G1 retargeted CSV joint motions before real-robot execution. |
| `SV_Soul` | AI-powered Stardew Valley NPC mod with character personas, persistent memory, game-state awareness, and native chat UI. |
| `TransVTLA-RealDataCollect` | Real-robot multimodal data collection and RLDS/TFDS data engineering toolkit for TransVTLA/VLA experiments. |
| `unitree_mujoco` | MuJoCo + Unitree SDK2 simulator for low-level control verification, G1 sim-to-real workflows, and keyframe playback. |
